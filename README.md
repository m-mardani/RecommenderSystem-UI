# RecommenderSystem-UI

رابط کاربری پیشرفته Next.js برای سیستم توصیه‌گر مبتنی بر FastAPI

## معرفی

این پروژه یک رابط کاربری کامل و حرفه‌ای برای سیستم توصیه‌گر است که با استفاده از فناوری‌های مدرن وب ساخته شده است.

## ویژگی‌های کلیدی

- ✅ **Next.js 16** با TypeScript
- ✅ **TailwindCSS** برای طراحی واکنش‌گرا
- ✅ **JWT Authentication** با تازه‌سازی خودکار توکن
- ✅ **Axios** برای مدیریت درخواست‌های HTTP
- ✅ **کنترل دسترسی مبتنی بر نقش** (ADMIN/USER)
- ✅ **پشتیبانی کامل از فارسی** (RTL + فونت Vazirmatn)
- ✅ **مدیریت حالت‌های بارگذاری و خطا**
- ✅ **صفحات کامل**: ورود، ثبت‌نام، داشبورد، داده‌ها، آموزش، پایش، مدل‌ها، توصیه‌ها

## شروع سریع

### پیش‌نیازها

- Node.js 18+
- npm یا yarn
- سرور FastAPI در حال اجرا

### نصب

```bash
cd frontend
npm install
```

### تنظیمات

فایل `.env.local` را ایجاد کنید:

```bash
cp .env.example .env.local
```

و آدرس API را تنظیم کنید:

```env
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

### اجرا

```bash
npm run dev
```

برنامه در آدرس [http://localhost:3000](http://localhost:3000) در دسترس خواهد بود.
